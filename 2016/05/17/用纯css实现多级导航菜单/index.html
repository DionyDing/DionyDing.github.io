用纯css实现多级导航菜单

css代码部分：


html,body,div,ul,li,a{margin:0;padding: 0;text-decoration: none;}
        .nav{
            font-size: 28px;
            color: mediumpurple;
            text-align: center;
        }
        .nav li{
           float:left;
           list-style: none;
           /relative为后续下级菜单absolute定义提供基位置/
           position: relative;
           width:250px;}
        .nav a:hover{
            background-color: mediumpurple;
            color: white;
        }
        .firstMenu li a,.secondMenu li a,.thirdMenu li a{
            display: block;
            background: yellowgreen;
            border:2px solid transparent;
            padding: 5px 9px;
            background-clip: padding-box;
        }
        .thirdMenu,.secondMenu{display: none;}
        .secondMenu{
            position: absolute;
            top:100%;/顶边与一级菜单底边对齐/
            left:0;
        }
        .thirdMenu{
            position: absolute;
            left: 100%;/左边与二级菜单右边对齐/
            top:0;
        }
        .secondMenu li,.thirdMenu li{
            float: none;
        }
        /控制下拉菜单悬停显示/
        .firstMenu li:hover > ul{
            display: block;
        }
        /记录选中的菜单栏/
        .nav li:hover>a{
            background-color: mediumpurple;
            color: white;
        }

html代码部分：

<div class=”nav”>
    <ul class=”firstMenu”>
        <!–一级开始–>
        <li><a href=”#”>power</a></li>
        <li><a href=”#”>money</a></li>
        <li><a href=”#”>love</a>
            <!–二级开始–>
            <ul class=”secondMenu”>
                <li><a href=”#”>sports star</a></li>
                <li><a href=”#”>movie star</a></li>
                <li><a href=”#”>rock star</a>
                    <!–三级开始–>
                    <ul class=”thirdMenu”>
                        <li><a href=”#”>123</a></li>
                        <li><a href=”#”>abc</a></li>
                        <li><a href=”#”>def</a></l>
                    </ul>
                    <!–三级结束–>
                </li>
            </ul>
            <!–二级结束–>
        </li>
        <li><a href=”#”>fame</a>
            <!–二级开始–>
            <ul class=”secondMenu”>
                <li><a href=”#”>sports star</a></li>
                <li><a href=”#”>movie star</a></li>
                <li><a href=”#”>rock star</a>
                    <!–三级开始–>
                    <ul class=”thirdMenu”>
                        <li><a href=”#”>123</a></li>
                        <li><a href=”#”>abc</a></li>
                        <li><a href=”#”>def</a></li>
                    </ul>
                    <!–三级结束–>
                </li>
            </ul>
            <!–二级结束–>
        </li>
        <!–一级结束–>
    </ul>
    </div>

说明：

1.<li>标签下的<a>标签设为display:block可以使文字部分及整个周围空白都可以点击，可点击范围不止局限于文字
2.background-clip：padding-box使背景色范围只到padding，通过把边框置为透明可以显示后层的背景
3.注释中内容的很重要
4.基本思想是鼠标悬停时显示下级菜单，否则隐藏。再结合position:absolute通过从当前父级<li>开始定位（父级position要设为relative）控制下级菜单的显示位置
